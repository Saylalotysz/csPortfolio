My Projects:

Project 2(fall 2021): Animated Jack o Lantern

This is my Advanced Computer Science scaling variable project. I made a jack o lantern that has variable that scale to the given pumpkim width and pumpkin height. I did this by creating the pumpkin in a 200 by 200 width and height, then scaled all the variables created after it. One struggle that I had with this project was that I did not know how to get the background to flicker. I used frame count to create a function called flicker, which is essentially a timer. Then created a boolean value of whether or not it was on, which then changed the r, g, and b values of the background.

Project 1(spring 2021): Gumball Catcher

This is my gumball catching project from ap computer science. The goal is to catch as many gumballs as possible! The catch is that the gumballs speed gets faster and faster as you go. One difficulty I had with this project was that I was unable to get the basket to catch the gumballs. I overcame this by having the y value equal the edge of the basket, and if it didnt, you lost!
